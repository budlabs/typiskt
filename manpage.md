`typiskt` - touchtype training for dirt-hackers

SYNOPSIS
--------
```text
typiskt [--difficulty|-d INT] [--corpus|-c WORDLIST] [--time|-t SECONDS] [--width|-w WIDTH]
typiskt --list|-l
typiskt --help|-h
typiskt --version|-v
```

DESCRIPTION
-----------
https://github.com/rr-/10ff length


OPTIONS
-------

`--difficulty`|`-d` INT  

`--corpus`|`-c` WORDLIST  

`--time`|`-t` SECONDS  

`--width`|`-w` WIDTH  

`--list`|`-l`  

`--help`|`-h`  
Show help and exit.

`--version`|`-v`  
Show version and exit.


ENVIRONMENT
-----------

`XDG_CONFIG_HOME`  

defaults to: $HOME/.config

DEPENDENCIES
------------
`bash`
